JavaScript Numbers


HexaDecimal

JavaScript intreprets numeric constants as hexadecimal if they are preceded by 0x
        
            By Default, JavaScript Displayes numbers as base 10 decimals.

            But you can use the toString() method to output numbers as base 16)hex), base 8(otcal), or base 2(inary)


Numbers can be objects
Normally JavaScript numbers are primitive values created from literals: var x = 123;

        But numbers can also be defined as objects with the keyword new :  var y=new numbers(123);
