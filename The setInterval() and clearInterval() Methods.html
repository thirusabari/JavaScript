The setInterval() Method

The setInterval() method repeats a given function at every given time interval.
        
            window.setInterval(function, milliSeconds); The window.setInterval() method can be written without the window prefix:

            The first parameter is the function to be executed. 

            The second parameter indicates the length of the time-interval between each execution.

            This example executes a function called "myTimer" once every second (like a digital match).

How to stop the Execution?

The clearInterval() method stops the executions of the functions specified in the setInterval() method.
        
        window.clearInterval(timerVariable);  

        The window.clearInterval() method can be written without the window prefix.

        The clearInterval() method uses the variable returned from the setInterval().

        myVar=setInterval(function,milliSeconds);
        clearInterval(myVar;)
